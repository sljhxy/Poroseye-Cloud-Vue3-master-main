## 前端运行

```bash
# 克隆项目
git clone https://github.com/sljhxy/Poroseye-Cloud-Vue3-master.git

# 进入项目目录
cd Poroseye-Cloud-Vue3-master

# 安装依赖
npm install

# 启动服务
npm run dev

# 构建测试环境 npm run build:stage
# 构建生产环境 npm run build:prod
# 前端访问地址 http://localhost:80
```
